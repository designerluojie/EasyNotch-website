隐私政策

最后更新：2026 年 7 月 22 日

本隐私政策适用于 EasyNotch macOS App Store 版本（以下简称“本应用”）。 我们重视你的隐私：本应用的核心功能在你的 Mac 本地运行，不会将你的剪贴板、文件或音乐播放信息上传至 EasyNotch 的服务器。

1. 本应用处理的信息

  • 本地功能数据：设置、剪贴板历史、暂存文件或文件夹的访问引用、番茄钟记录，以及 AI 对话的本地界面数据，会保存在你的设备上，以实现相应功能。
  • 剪贴板内容：在你开启并使用剪贴板历史功能期间，本应用会读取系统剪贴板，并在本地保存文字、富文本、图片、SVG、文件或文件夹引用等内容。你应避免将不希望留存在本机的敏感信息复制到剪贴板。
  • 音乐播放信息：在你使用音乐模块时，本应用会在获得 macOS 自动化授权后，读取 Apple Music 或 Spotify 的当前播放状态、歌曲名称、艺人、专辑、进度和封面，用于在刘海面板显示和提供播放控制。该信息不会发送给 EasyNotch。

2. AI Chat 与第三方 AI 服务

AI Chat 是可选功能。只有当你自行配置 API 密钥并主动发送请求时，本应用才会将你的提问、对话内容及你选择附带的图片或其他输入，直接发送给你选择的 AI 服务提供商（目前可能包括 DeepSeek、通义千问、OpenAI 或 Google Gemini）。这些服务将依其各自的隐私政策处理该等信息。

EasyNotch 不提供 AI 中转服务器，也不会代为接收或保存你的 API 密钥。API 密钥储存在 macOS Keychain 中；你可在本应用的 AI 配置中删除它们。

3. 网络连接、分析与追踪

本应用的 App Store 版本不使用行为分析或跨应用追踪。使用 AI Chat 时会连接至你自行选择的 AI 服务；Spotify 的歌曲封面可能会从 Spotify 提供的 HTTPS 地址下载，仅用于本地展示。

我们不会出售你的个人信息，也不会将本应用中的数据用于广告定向或跨应用追踪。

4. 权限说明

  • 剪贴板：用于保存和再次使用剪贴板历史。
  • 文件访问：仅用于你主动拖入、选择或再次导出的文件及文件夹。
  • 自动化（Apple Events）：仅用于与 Apple Music 和 Spotify 交互以显示播放信息及执行播放控制。

你可在 macOS“系统设置”中管理本应用的相关权限；拒绝某项权限只会影响对应功能。

5. 信息保存与删除

本地数据会在实现功能所需的期间保留。你可以在本应用内清理剪贴板历史、移除暂存文件和删除 AI API 密钥；对于其他本地应用数据，也可以通过卸载本应用或清除其本地数据进行处理。请注意，原始文件是否可用也取决于它是否仍保存在你的设备中。

6. 数据安全与第三方服务

我们采用系统提供的本地存储与 Keychain 机制保护数据。通过 AI Chat 使用第三方服务时，数据安全、数据保存地点和保留期限由相应服务提供商决定；请在使用前阅读其适用的隐私政策，并根据你的所在地法规和需求自行判断是否使用。

7. 本政策的更新与联系我们

当功能或数据处理方式发生实质变化时，我们会在本页面更新本政策，并调整“最后更新”日期。如对隐私问题有任何疑问、意见或请求，请联系：easynotch@163.com。
